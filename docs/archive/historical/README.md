**Original:** ROADMAP.md
**Archived:** 2025-11-19
**Reason:** This roadmap was written when the project was still called "Creative Computation DSL" and planned a path from v0.2.2-mvp through v1.0.0 over a ~1 year timeline. These documents are archived rather than deleted because they:
1. Provide historical context for the evolution of the project
2. Document design decisions and rationale that may still be relevant
3. Show the progression from early concepts to current implementation
4. May contain useful ideas for future development

However, **do not rely on these documents for current project information** - they are historical snapshots only.
